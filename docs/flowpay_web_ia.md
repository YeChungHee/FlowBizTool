# FlowPay Web IA

## 목표
플로우페이 세일즈 자동화 시스템을 `거래 가설 평가 + 제안서 생성 + 이메일 생성 + 검색 재활용`까지 한 흐름으로 연결한 웹 앱으로 설계한다.

핵심 UX 원칙:
- Apple 스타일의 단순한 정보 구조
- 큰 제목, 넓은 여백, 카드 중심 레이아웃
- 초등학생도 이해할 수 있는 쉬운 용어
- 한 화면에 한 가지 목적만 보이게 구성
- 단계형 입력 위저드로 입력 부담 최소화

## 용어 기준
- `신청업체`: 플로우페이를 신청한 회사
- `매입처`: 제품이나 원자재를 공급하는 회사
- `매출처`: 신청업체가 납품하고 돈을 받는 회사
- `결제기간`: 돈을 먼저 넣은 날부터 다시 받는 날까지
- `한도액`: 이번에 진행 가능한 최대 매입액
- `마진율`: 위험을 반영해 붙는 이익 비율
- `마진액`: 신청금액 x 마진율
- `매입액`: 이번에 먼저 지급할 금액
- `매출액`: 매출처에서 다시 받는 금액

## 최상위 정보구조
1. 홈
2. 새 거래 검토 시작
3. 거래 가설 평가 결과
4. BizAiPro 제안서
5. 이메일 만들기
6. 자료 검색
7. 설정

## 화면 목록

### 1. 홈
목적:
- 오늘 처리할 거래 검토 건을 한눈에 본다.
- 진행 중인 건과 막힌 건을 바로 찾는다.

주요 영역:
- 상단 요약 카드: `오늘 신규`, `검토 필요`, `승인 대기`, `보완 요청`
- 최근 거래 검토 리스트
- 최근 생성 문서: 제안서, 이메일
- 빠른 실행 버튼: `새 거래 검토`, `제안서 만들기`, `자료 검색`

### 2. 새 거래 검토 시작
목적:
- 영업 참고용 거래 평가 입력을 단계별로 쉽게 받는다.

하위 단계:
1. 신청업체 정보
2. 거래 구조
3. 결제 정보
4. 상대 회사 정보
5. 공식 서류 확인
6. 최종 확인 후 평가 실행

### 3. 거래 가설 평가 결과
목적:
- 결과를 숫자보다 다음 영업 행동 중심으로 보여준다.

주요 영역:
- 결과 배지: `거래 가능성 높음`, `추가 확인 필요`, `제안 비추천`
- 핵심 수치: `한도액`, `마진율`, `마진액`, `결제기간`
- 위험요인 카드
- 월별 결제위험 그래프
- 추가 확인이 필요한 자료 목록
- 다음 액션 버튼

### 4. BizAiPro 제안서
목적:
- 심사 결과를 고객 설명용 문서로 변환한다.

주요 영역:
- 업체 개요
- 거래 구조 설명
- 제안 조건
- 기대 효과
- 추가 요청 자료
- 문안 수정 패널

### 5. 이메일 만들기
목적:
- 제안서와 심사 상태를 기반으로 이메일을 자동 작성한다.

주요 유형:
- 승인 안내
- 조건부 승인
- 보완 요청
- 진행 어려움

### 6. 자료 검색
목적:
- 기존 상담, 리포트, 제안서, 이메일을 재활용한다.

주요 기능:
- 회사명 검색
- 업종별 검색
- 유사 거래 검색
- 과거 승인/거절 사례 검색

### 7. 설정
목적:
- API, 템플릿, 사용자 기본값을 관리한다.

주요 기능:
- ECOS / DART / 추후 외부연동 키 관리
- 제안서 템플릿 관리
- 이메일 문구 기본값 관리

## 사용자 흐름
1. 홈에서 `새 거래 검토`를 누른다.
2. 신청업체, 매입처, 매출처, 결제기간, 매입액을 입력한다.
3. 서류를 업로드하고 자동확인 결과를 본다.
4. 평가를 실행한다.
5. 결과 화면에서 `예상 한도액`, `예상 마진율`, `예상 마진액`, `위험요인`을 확인한다.
6. 바로 제안서를 만든다.
7. 제안서를 바탕으로 이메일을 만든다.

## 내비게이션 원칙
- 좌측 사이드바: 상위 메뉴 고정
- 상단 바: 현재 심사건 이름, 상태, 저장 여부 표시
- 본문: 단일 목적 레이아웃
- 하단 CTA: `이전`, `저장`, `다음`, `심사 실행`

## 시각 스타일 방향
- 배경은 따뜻한 화이트와 연한 그레이 중심
- 카드 단위 정보 배치
- 상태색은 과하지 않게 사용
- 성공은 녹색, 검토는 오렌지, 위험은 레드 계열
- 폰트는 `Apple SD Gothic Neo`, 보조로 `SF Pro` 계열 가정

## 반응형 기준
- 데스크톱 우선 설계
- 태블릿에서는 2열 카드 유지
- 모바일에서는 위저드와 결과 카드 중심 단일 열
